minikube
========

# Cheatsheet

## Start the cluster

```minikube start```

## Get all pods of the cluster

```kubectl get po -A```

## Get the dashboard

```minikube dashboard```

## Pause/Unpause the cluster

This will pause all services without deployments

```minikube pause```

```minikube unpause```

## Stop the cluster

```minikube stop```

## Delete the cluster

```minikube delete --all```
